Email Classification using Machine Learning

Spam emails are considered one of the biggest challenges for the Internet. Thus email classification, which aims to correctly classify legitimate and spam emails, becomes an important topic for both industry and academic purposes. To achieve this goal, Rule-based techniques have been extensively applied to this field. we find that rule-based email Classification is acceptable by users in real email classification. However in recent, several studies reveal that rule-based classifiers suffer from some limitations such as efficiency fluctuation, Since these approaches are based on standard rules, they usually can be easily bypassed by robot software with new domains. hence many works start focusing on designing more complex methods of identifying spam from other emails that is using machine learning algorithms. That is extensive machine learning algorithms such as Support Vector Machine, k-means classifiers, and many other methods can be used for email classification, which produces better accuracy and performance than rule-based Classifiers. We used a Support Vector Machine (SVM) for our project as the Classification model.
